PythOS

專為每個人設計的輕量級、基於 Linux 和 Python 的作業系統。

下載 加入 Discord 查看原始碼

輕量且高效

PythOS 移除了不必要的組件，使用最快速的 Linux 系統為基礎。專注於提供一個輕量高效的作業系統環境，即使在較低配置的硬體上也能順暢運行。

強大的 Python 支援

預安裝最新版本的 Python 和常用函式庫，在優化環境中支援快速開發和部署 Python 應用程式。

成為應用開發者！

使用 Python 創建您自己的應用！只需在 bin 資料夾中添加一個與 .py 檔同名的資料夾（不含 .py 副檔名），就可以開始了！

自動化與部署

內建腳本和自動化工具，簡化系統管理和應用部署過程，支援容器化和虛擬化技術。

可自訂

彈性的設定選項讓您能夠根據需求客製化系統環境，從輕量裝置到高效能伺服器都能滿足需求。

即時啟動！

體驗 PythOS 的靈活性，使用即時啟動功能直接從 USB 或 DVD 執行系統，無需安裝。完美適合測試、故障排除或隨時攜帶的運算需求。

介紹

PythOS 是一個輕量級的基於 Linux 和 Python 的作業系統，專為每個人設計。它提供了一個極簡而強大的環境。

安裝

請依照以下步驟安裝 PythOS：

    從下載獲取最新的 ISO 檔案。
    使用工具如 dd 或 Rufus 製作啟動 USB。
    從 USB 啟動電腦
    在 SystemBack 啟動載入器中，您可以選擇：

     1. 選項 1：如果您希望直接執行系統（選擇此選項後可跳過後續指示）

     2. 選項 2：如果您希望將系統安裝到磁碟（繼續執行後續指示）

    選擇要安裝的目標儲存裝置（注意！這將清除所有資料）
    按照螢幕上的指示操作
    開始享用！
